# RememberMe Chrome Extension

## Setup Instructions
1. Download the latest version of the RememberMe Chrome extension from [GitHub Releases](https://github.com/owner/repo/releases).
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable "Developer mode" by toggling the switch in the top right corner.
4. Click on "Load unpacked" and select the folder containing the downloaded extension files.
5. The RememberMe extension should now appear in your extensions list and be ready for use.

## Features
- Securely save and remember passwords and user credentials.
- One-click autofill for login forms and fields.
- Support for multiple accounts and profiles.
- User-friendly interface with customizable options.
- Regular updates for improved functionality and security.

## Architecture Overview
- **Frontend**: Built with HTML, CSS, and JavaScript to create a seamless user experience within the Chrome browser.
- **Background Script**: Manages data storage and retrieval, API requests, and event handling in the background.
- **Content Script**: Interacts with web pages to autofill fields and capture user credentials.
- **Options Page**: Provides settings for users to customize their extension experience.

## Usage Guide
1. Once the extension is installed, click on the RememberMe icon in the toolbar.
2. Create a new account or log in to save your credentials.
3. Navigate to a website where you need to enter your credentials.
4. Click on the RememberMe icon again to autofill the login form.
5. You can manage saved accounts through the options page or edit credentials as needed.
